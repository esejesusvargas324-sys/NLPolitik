Base Metodológica: El Proyecto MARPOR

Manifesto Project (MARPOR)

NLPolitik se construye sobre los cimientos del Manifesto Project (MARPOR), el estándar académico para el análisis comparativo de programas electorales. Este proyecto ha codificado sistemáticamente miles de manifiestos en más de 67 países, demostrando que las ideologías políticas pueden ser categorizadas de manera estructurada y empírica (Lehmann et al., 2025).

De MARPOR a NLPolitik: Un Puente Metodológico

El corpus y el esquema de clasificación de MARPOR proporcionan la base de conocimiento necesaria para entrenar el clasificador supervisado (SVM) de NLPolitik. Mientras MARPOR se enfoca en el análisis manual de manifiestos, NLPolitik traslada ese rigor al análisis automatizado y escalable del discurso político digital, utilizando la escala RILE como puente para la clasificación ideológica en el eje izquierda-derecha.
